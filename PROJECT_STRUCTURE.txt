File Descriptions
=================

ğŸ¤– AGENTS (The AI Brain)
-------------------------
base_agent.py (72 lines)
  - Abstract base class for all agents
  - Handles Gemini API calls
  - JSON parsing utilities
  - Error handling

chemistry_agent.py (124 lines)
  - Converts molecule names to SMILES
  - Calculates molecular properties (MW, LogP, TPSA, etc.)
  - Analyzes drug-like characteristics
  - Provides application information

synthesis_agent.py (93 lines)
  - Plans industrial synthesis routes
  - Identifies raw materials and intermediates
  - Assesses synthesis complexity
  - Provides step-by-step procedures

market_agent.py (141 lines)
  - Estimates raw material prices (Iranian market)
  - Projects market trends (2023-2027)
  - Calculates total production costs
  - Global vs Middle East market analysis

feasibility_agent.py (150 lines)
  - Loads factory equipment inventory
  - Assesses production feasibility
  - Identifies missing equipment
  - Provides optimization recommendations

chat_agent.py (115 lines)
  - Handles conversational interactions
  - Maintains chat context
  - Formats molecule data for AI
  - Provides intelligent responses
